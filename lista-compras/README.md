# 🛒 App Lista de Compras

**Desarrollado por:** Dylan Ariel Aragón

Una aplicación móvil minimalista y rápida construida con React Native, Expo y Bun. Permite a los usuarios gestionar sus compras diarias de manera eficiente, agregando productos, marcándolos como obtenidos y eliminándolos.

## 📱 Funcionalidades (Cómo usar la app)
* **Agregar productos:** Escribe el nombre de un artículo en el campo de texto superior y presiona el botón **"Agregar"** o la tecla "Enter/Done" en tu teclado.
* **Marcar como comprado:** Toca una vez (toque rápido) sobre cualquier ítem de la lista para tacharlo y marcarlo con un `✔`. Toca nuevamente para deshacer la acción.
* **Eliminar productos:** Mantén presionado (toque largo) sobre cualquier ítem de la lista para eliminarlo permanentemente.

## 🛠️ Requisitos Previos del Sistema
Para ejecutar este proyecto en tu entorno local, necesitas tener instalado:
1. [Node.js](https://nodejs.org/) (Versión LTS recomendada)
2. [Bun](https://bun.sh/) instalado en tu sistema (`curl -fsSL https://bun.sh/install | bash` en Linux/macOS).
3. Aplicación **Expo Go** instalada en tu dispositivo móvil (disponible en Android e iOS).
